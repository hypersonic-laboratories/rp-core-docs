# Functions
This is a class used to manage and maintain multiple different points. Allowing tracking players interactions easier.


### GetConfig()
Sets the marker as visible.

```lua
local cfg = Core.GetConfig()
```

### Core.RegisterHUD(...args)
name, location, closeEvent
```lua
local hud = Core.RegisterHUD('hud', './ui/main.html', 'closeEvent')

Events.Subscribe('closeEvent', function()
    -- Ran when the hud is closed
end)
```

### Core.GetHUD(name)
```lua
local hud = Core.GetHUD('hud')

-- Logic here
```

### Core.PlayAnimation(...args)
anim, slot_flag, loop, blend_in_time, blend_out_time, play_rate, stop_all_montages
```lua
Core.PlayAnimation("griddy", AnimationSlotType.FullBody, false, 0.25, 0.25, 1.0, false)
```

### Core.StopAnimation(anim)
```lua
Core.StopAnimation("griddy")
```
