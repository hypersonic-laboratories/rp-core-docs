# Vehicle Class
Vehicle Class used to add additional data to each player vehicle, such as fuel level, VIN number etc.
<Callout type="info" emoji="ℹ️">
    Note: This is the vehicle CLASS, when we refer to vehicle here we aren't referring to HELIXVehicle rather a vehicle data class, similar to the player class.

    ```lua copy
    local vehicle = Core.GetVehicleData(vehicleEntity)

    -- vehicle.vin <- have access to this now
    ```
</Callout>

| location    |  Vector   |   true |
