# Server
Player Class with functions used to assist

## Methods
### revive(location?)
Revives the player, if location is given it will revive them at the coordinates
```lua
local xPlayer = Core.GetPlayerFromId(player:GetID())

xPlayer.revive()
```

### slay()
Slays the player, instantly killing them on the spot
```lua
local xPlayer = Core.GetPlayerFromId(player:GetID())

xPlayer.slay()
```

### save ()
Saves the players savable data to the database.
```lua
local xPlayer = Core.GetPlayerFromId(player:GetID())

xPlayer.slay()
```

### getXP()
Returns the players current XP level.
```lua
local xPlayer = Core.GetPlayerFromId(player:GetID())

local xp = xPlayer.getXP()
```

### getInventoryTarget()
Returns the players current inventory target.
```lua
local xPlayer = Core.GetPlayerFromId(player:GetID())

local inventoryTarget = xPlayer.getInventoryTarget()
```

### getVehicle()
Returns the vehicle the player is currently in, if none it will return nil.
```lua
local xPlayer = Core.GetPlayerFromId(player:GetID())

local vehicle = xPlayer.getVehicle()
```

### get(key)
Returns a stored key value on the player class, this can be used for dynamic variables.
```lua
local xPlayer = Core.GetPlayerFromId(player:GetID())

xPlayer.set('foo', 'bar')

local x = xPlayer.get('foo') -- x = 'bar'
```
